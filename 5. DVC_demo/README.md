<h3>To initialize DVC</h3>
<pre><code>dvc init</code></pre>

<h3>To add file through DVC</h3>
<pre><code>dvc add 'File Name or File Path'</code></pre>

<h3>To switch to different version of data</h3>
<pre><code>Step 1: git checkout '#committed code id'</code></pre>
<pre><code>Step 2: dvc checkout</code></pre>
